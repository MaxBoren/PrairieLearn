Questions now have the ability to be nested inside of subfolders — meaning that instead of a sprawling questions folder inside of your course, you can now organize these into folders however you wish!

The full QID of the question is now the path from the questions folder. For example, the demoAnsiOutput folder above would be demoQuestions/demoAnsiOutput when creating assessments.
